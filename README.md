# Homebrew STCLab

Homebrew tap for STCLab internal tools.

## Setup

```bash
brew tap STCLab-Inc/stclab
```

## Available Casks

| Cask | Description |
|------|-------------|
| `amaranth-check` | macOS menu bar app for Amaranth attendance tracking |

```bash
brew install --cask STCLab-Inc/stclab/amaranth-check
```

## Updating

```bash
brew update && brew upgrade --cask amaranth-check
```

## Contributing

1. Create a branch
2. Add or modify cask in `Casks/`
3. Open a PR
